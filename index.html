Projetos

Churrascômetro

O Churrascrômetro é um site que ajuda a calcular a quantidade de comida e bebida necessária para um churrasco. Basta informar o número de convidados e o site fornece uma lista detalhada do que será necessário, incluindo a quantidade de carnes, acompanhamentos, bebidas e outros itens.

Tour World

O Tour World é um site abrangente de planejamento de viagens que oferece aos usuários uma ampla gama de opções de destinos e atividades de turismo em todo o mundo. Com um foco em experiências autênticas e personalizadas, o Tour World ajuda os viajantes a descobrir locais impressionantes.

Richard Construções

Richard Construções é um site completo e confiável para encontrar todos os materiais de construção que você precisa. Com uma ampla variedade de produtos de alta qualidade, desde tijolos e cimentos até ferramentas e acabamentos, o site oferece uma experiência conveniente para profissionais da construção e consumidores em geral.

Lista de tarefas

O site de lista de tarefas desenvolvido com React e TypeScript é uma aplicação web que permite aos usuários criar, organizar e gerenciar suas tarefas de forma eficiente. Com uma interface intuitiva e responsiva, os usuários podem adicionar novas tarefas, marcar como concluídas e definir prioridades.

eFood

Bem-vindo ao eFood, o seu destino gastronômico online! O eFood é um site de restaurante que reúne uma ampla variedade de opções culinárias em um só lugar, oferecendo uma experiência de entrega de comida conveniente e deliciosa. Com um simples clique, você terá acesso a uma infinidade de restaurantes locais e pratos saborosos, desde cozinha internacional até comida caseira reconfortante.

Calculadora

O site de calculadora em tempo real desenvolvido com Vue é uma aplicação web interativa que permite aos usuários realizar cálculos matemáticos de forma rápida e precisa.
